# Intent Solutions Homebrew Tap

Homebrew formulae for Intent Solutions tools.

## Installation

```bash
brew tap intent-solutions-io/tap
brew install gvid
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `gvid` | Mission Control dashboard for Gastown multi-agent workspaces |

## Usage

```bash
# Start daemon
gvid

# With custom port
gvid --port 8080

# TUI client
gvi-tui
```
